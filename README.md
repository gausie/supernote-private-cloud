# supernote-private-cloud
You can point portainer here to get supernote private cloud running easily

```bash
MYSQL_ROOT_PASSWORD=
MYSQL_DATABASE=
MYSQL_USER=
MYSQL_PASSWORD=
REDIS_PASSWORD=
REDIS_HOST=
DB_HOSTNAME=
# These images don't have `latest` tag. These are the latest at the time of writing
NOTELIB_IMAGE_TAG=6.9.3
SUPNOTE_IMAGE_TAG=26.01.21
```
